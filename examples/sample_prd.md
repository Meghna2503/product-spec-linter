# PRD: Checkout Redesign v2.1
**Author:** Product Team  
**Status:** Draft  
**Last Updated:** April 2026

---

## Overview
We want to make the checkout experience better and faster for our users.
The new checkout should be seamless and intuitive across all devices.

---

## Goals
- Improve checkout conversion
- Reduce cart abandonment
- Support all popular payment methods
- Make it work on major browsers

---

## User Stories

### US-01: Guest Checkout
As a shopper, I want to checkout without creating an account,
so that I can complete my purchase quickly.

### US-02: Saved Addresses
As a registered user, I want my addresses saved automatically,
so that repeat checkout is easy.

### US-03: Payment
As a shopper, I want to pay using my preferred payment method.
The system should process payments quickly.

### US-04: Order Confirmation
As a shopper, I want to receive confirmation after purchase,
so that I know my order was placed.
All users must be logged in before placing an order.

### US-05: Mobile Checkout
As a mobile shopper, I want a smooth checkout experience on my phone.
